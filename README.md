# Multi-Tool Agentic RAG Chatbot

An AI-powered chatbot built using LangGraph that supports:

- PDF-based Retrieval Augmented Generation (RAG)
- Multi-tool calling (Web search, Calculator, Stock API)
- Persistent multi-thread chat using SQLite
- FAISS vector store
- HuggingFace embeddings
- Streamlit interactive UI

## Tech Stack
LangGraph, LangChain, Groq (LLaMA 3.1), FAISS, HuggingFace, Streamlit

## Features
- Upload PDF and chat with it
- Tool-calling agent architecture
- Persistent conversation threads
- Streaming responses
